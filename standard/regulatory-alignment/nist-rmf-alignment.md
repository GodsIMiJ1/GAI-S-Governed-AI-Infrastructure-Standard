# NIST AI RMF Alignment

**Document ID:** GAIS-REG-003  
**Version:** 1.0  
**Status:** Informative Mapping

## 1. Purpose

This document maps GAI-S controls to NIST AI RMF functions (GOVERN, MAP, MEASURE, MANAGE).

## 2. Function Mapping

| NIST AI RMF Function | Relevant GAI-S Controls | Typical Evidence |
|---|---|---|
| GOVERN | `GAIS-CTRL-AS-01`, `GAIS-CTRL-AS-02`, `GAIS-CTRL-DG-02`, `GAIS-CTRL-AL-02`, `GAIS-CTRL-IR-02` | Governance charters, role maps, retention policy, CAPA records |
| MAP | `GAIS-CTRL-RC-01`, `GAIS-CTRL-DG-01` | Risk context docs, data provenance analysis |
| MEASURE | `GAIS-CTRL-RC-02`, `GAIS-CTRL-AL-01` | Risk band metrics, audit logs, threshold validation tests |
| MANAGE | `GAIS-CTRL-DC-01`, `GAIS-CTRL-DC-02`, `GAIS-CTRL-IR-01` | Containment controls, incident playbooks, response traces |

## 3. Notes

- This mapping assists evidence organization for AI risk management programs.
- Organizations SHOULD tailor mappings to use case, impact profile, and sector obligations.
